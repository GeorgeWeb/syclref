# operator!=

---

```cpp
bool operator!=(const queue &lhs, const queue &rhs)
```


Determines if lhs and rhs are not equal. 


---
**Parameters:**

 - `queue lhs`
: Left-hand-side object in comparison 

 - `queue rhs`
: Right-hand-side object in comparison 


---
**Returns:** True if different underlying objects
